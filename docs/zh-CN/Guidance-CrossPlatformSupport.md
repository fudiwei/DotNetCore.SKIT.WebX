### 跨平台支持

**STEP.WebX** 包含多个版本的库：

* .NET Core 2.1
* .NET Core 2.2
* .NET Core 3.0
* .NET Core 3.1
* .NET 5.0

这意味着你可以在如下环境使用 **STEP.Locking**：

* 在 Windows 中使用 .NET Core；
* 在 Windows 中使用 .NET 5；
* 在 Linux 或 MacOS 中使用 .NET Core；
* 在 Linux 或 MacOS 中使用 .NET 5；
* 任何支持 Mono 5、.NET Core 2、.NET Core 3 或 .NET 5 的其他平台。
